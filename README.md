Auto-Mate your brew.

First Project: Brew-Mate Hydrometer based on rp2040 microcontroller. 

    Prototype v 0.0.1 : Raspberry Pi Pico + ESP32-S2-Wroom (or mini) + GY-LSM6DS3
              Feature : - Wifi
                        - Bluetooth
                        - Temperature
                        - Tilt Angle 

Future Project : - Dashboard
